# Divide
The divide command divides the first input number with the second input number
```
div [number, number]
```
---
## Overview
```yaml
name: 'div'
min: 2
max: 2
arguments:
    - number
    - number
return: number
```
---
## Example 
```
>>> div 100 20
5
```
