# Replace
The multiply command returns the product of the numbers passed
```
mult [number, number, number, number]
```
---
## Overview
```yaml
name: 'mult'
min: 2
max: 4
arguments:
    - number
    - number
    - number
    - number
return: number
```
---
## Example 
```
>>> mult 2 24
48
```
